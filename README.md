# Mi Portafolio Personal 🚀

¡Bienvenido al repositorio de mi portafolio web! Este proyecto es una vista de mi trayectoria académica, habilidades técnicas y los proyectos que he desarrollado como estudiante de programación.

## 📱 Vista Previa
El portafolio presenta un diseño moderno y oscuro (Dark Mode), optimizado para una lectura clara y una navegación fluida. Incluye secciones de:
- **Hero Section:** Presentación principal con foto de perfil.
- **Sobre Mí:** Resumen de mi perfil técnico y aspiraciones profesionales.
- **Habilidades:** Visualización de competencias técnicas mediante barras de progreso.
- **Proyectos:** Galería de trabajos destacados con enlaces directos a sus repositorios.
- **Contacto:** Enlaces directos a mis redes profesionales.

## 🛠️ Tecnologías Utilizadas
Para la construcción de este sitio utilicé un stack centrado en el rendimiento y la estética:

* **HTML5:** Estructura semántica del sitio.
* **CSS3:** Diseño personalizado utilizando variables (`:root`), Grid Layout y Flexbox para asegurar la responsividad.
* **JavaScript:** Implementación de interactividad, como el ocultamiento dinámico del header al hacer scroll.
* **FontAwesome:** Iconografía profesional para habilidades y redes sociales.
* **Google Fonts:** Tipografías legibles y modernas.

## 🚀 Proyectos Destacados en el Portafolio
Dentro del sitio podrás encontrar detalles sobre:

2.  **NeoStock:** Sistema de gestión de inventario robusto para consola desarrollado en C#.
